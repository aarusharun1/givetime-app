# GiveTIme

Official repository for the GiveTime web-based application. GiveTime is a volunteer opportunity platform that connects people with organizations in their community. GiveTime was created with the intention of making community service more accessible and straightforward. Browse by category, find opportunities that match your interests, and start giving back today. 

# About

GiveTime was created by Aarush Arun, a student at Novi High School in southeast Michigan, to solve a simple problem: finding the right volunteer opportunity shouldn't be hard. Search results are often misleading, finding organizations that are right for every individual takes time, and there's no single place to see what's actually available nearby.

This platform organizes volunteer organizations into clear categories so users can quickly find opportunities that fit what they're looking for.

# Tech Stack
 
- Next.js 15 (React 19, TypeScript, App Router)
- Tailwind CSS
- Supabase (auth, database)
- Vercel (hosting)

# Features

- Browse 100+ verified volunteer organizations across Southeast Michigan
- Filter by category, format (in person / online), county, and age eligibility
- User accounts (email/password and Google sign-in)
- Volunteer hour logging with date, duration, and notes
- Analytics dashboard with breakdowns by category, organization, and month
